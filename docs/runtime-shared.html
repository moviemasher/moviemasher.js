runtime-shared

The @moviemasher/runtime-shared module is imported by high-level libraries like @moviemasher/lib-client and @moviemasher/lib-server, and not typically installed directly. It exports core types, interfaces, and utility functions needed at runtime by all other modules.

This module is built from code and configuration available in the Movie Masher Github Repository. The NPM distribution includes TypeScript source code files, as well as a
minified JavaScript file intended to be imported directly from other modules. All source files are distributed as a single JavaScript file. The JavaScript includes source maps to the TypeScript for easier debugging.
